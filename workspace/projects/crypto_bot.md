# Crypto Futures Bot

## Objetivo
Bot automatizado para trading de futures com IA multi-agente.

## Stack
- Python + CCXT
- LangChain / LangGraph
- Binance API
- PostgreSQL (histórico de trades)

## Agentes Planejados
1. **Market Scanner** — escaneia oportunidades 24/7
2. **Signal Generator** — analisa e gera sinais
3. **Risk Manager** — valida risco antes de executar
4. **Executor** — executa ordens na Binance
5. **Reporter** — envia relatórios no Telegram
